## [1.0.0] — 2026-03-29

### Added
- `zca_recon` — Common Area reconciliation engine
  - `run_reconciliation()` — full flow with CSV import option
  - `export_update()` — export Discrepancy/Partial rows
  - `export_add()` — export Not Found in CSV rows
  - Cross-platform file dialogs (Mac + Windows)
  - Auto-coloring of status column
  - Dashboard timestamp stamping
  - Package field sync with Zoom Meetings stripping
- Windows and Mac one-time setup scripts with daily auto-update
- Full documentation in docs/overview.md
